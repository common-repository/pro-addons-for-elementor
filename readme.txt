Pro Addons For Elementor is an essential addon for Elementor that provides the Elementor Pro features and functionality for free.

== Description ==

Enhance your [Elementor](https://wordpress.org/plugins/elementor/) page-building experience with Pro Addons For Elementor. Add powers to your website builder using our easy-to-use theme builder, which can be used to make theme templates such as single, archive, search results, etc. Pro Addons For Elementor provides lots of widgets that can be used to enhance your website.


## Requirements

- WordPress (Version 5.0 or higher)
- Elementor

## ℹ️ Pro Addons For Elementor Features

### Theme Builder

Pro Addons For Elementor theme builder can be used to make the theme templates for the below categories.

- **Single Post Template**
- **Taxonomy Template**
- **404 Template**
- **Search Template**


### Widgets (Free)
- **Posts:** Showcase your blog posts in your website styles.
- **Flip Box:** Use animated Flip Boxes to highlight any content.
- **Contact Form 7:** Customize your forms without any coding.
- **Lottie:** The Lottie animation widget can be used to add the animation from the JSON file to your website.
- **Post Title:** The Post Title widget can be used to show the current post title for built-in and custom post types.
- **Post Content:** The Post Content widget can be used to show the content of the current post in the Theme Builder template.
- **Featured Image:** Show the featured image of your post in the theme builder template with ease.
- **Post Info/Meta:** The Post Info widget can be used to show the meta information of the single post such as author, date, categories, comments count, etc.
- **Post Navigation:** The Post Navigation widget provides the navigation button for the single post to go to the next and previous posts.
- **Term Title:** Show the current Taxonomy term title in the theme builder template.
- **Term Description:** Show the current Taxonomy term description in the theme builder template.
- **Post Meta Text:** Use this widget to show the text data of your custom field/meta field in the pages and template.
- **Post Meta Image:** Use this widget to show the images from your custom field/meta field in the pages and template.

### Additional Features

- **Custom CSS** - This feature is available in the Elementor builder page setting and can be used to add page-wise custom CSS for the pages, posts, and templates.
- **Header Code** - Header code feature can be used to add custom code in the <head> tag of the plugin, this is also available in the Elementor builder page setting.
- **Footer Code** - This can be used to add custom code in the page before the end of </body> tag, this is also available in the Elementor builder page setting.



== Installation ==

Install the plugin from the WordPress admin page "Plugins > Install"

or
	
1. Upload `pro-addons-for-elementor` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
	

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =

No. You cannot use without Elementor since it's an addon for Elementor.

= Does it work with any theme? =
	
Absolutely! It will work with any theme where Elementor works.
	
= How much performance does Pro Addons For Elementor need? =

Actually none! We designed it with maximum page performance in mind. Our tests did show literally no change in page loading speed.

= Is Pro Addons For Elementor compatible with Caching plugins? =
	
Yes, absolutely! There is no dynamic logic in the PHP code like other plugins have.
